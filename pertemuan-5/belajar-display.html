🎓 Belajar CSS Display Properties

Display menentukan bagaimana elemen ditampilkan di halaman (block, inline, inline-block, none, flex, grid)

1️⃣ DISPLAY: BLOCK

Pengertian Block

display: block membuat elemen mengambil seluruh lebar yang tersedia (100%). Elemen block selalu dimulai dari baris baru dan memaksa elemen berikutnya ke baris baru juga.

display: block;

Karakteristik Block:

  • ✅ Mengambil 100% lebar (atau sesuai parent width)
  • ✅ Selalu dimulai baris baru
  • ✅ Margin & padding bekerja di semua sisi
  • ✅ Width & height bisa diatur
  • ❌ Tidak bisa di-align horizontally dengan elemen lain di baris yang sama

Contoh Element Block Default:

Tag-tag ini secara default adalah block:

<div>, <p>, <h1>-<h6>, <header>, <footer>, <nav>, <section>, <article>, <ul>, <ol>, <li>

Demo Visualisasi:

Lihat? Setiap box mengambil 100% lebar dan dimulai dari baris baru!

Kode HTML:

<div class="block-demo">Box 1</div> <div class="block-demo">Box 2</div> <div class="block-demo">Box 3</div> <style> .block-demo { display: block; background: #3498db; padding: 20px; margin-bottom: 15px; } </style>

2️⃣ DISPLAY: INLINE

Pengertian Inline

display: inline membuat elemen hanya mengambil lebar yang dibutuhkan. Elemen inline bisa berdampingan dengan elemen lain di baris yang sama, seperti teks biasa.

display: inline;

Karakteristik Inline:

  • ✅ Hanya mengambil lebar sesuai konten
  • ✅ Bisa berdampingan dengan elemen lain di baris sama
  • ✅ Tidak ada baris baru sebelum/sesudah
  • ❌ Width & height TIDAK bisa diatur
  • ❌ Margin-top & margin-bottom tidak bekerja
  • ⚠️ Padding bekerja tapi bisa overlap elemen lain

Contoh Element Inline Default:

Tag-tag ini secara default adalah inline:

<span>, <a>, <strong>, <em>, <img>, <button>, <input>, <label>

Demo Visualisasi:

display: inline
Ini adalah teks normal Box 1 (inline) dilanjut dengan teks Box 2 (inline) dan lagi teks Box 3 (inline) semuanya di baris yang sama!
Lihat? Semua box berada di baris yang sama, berdampingan seperti teks!

Kode HTML:

Teks normal <span class="inline-demo">Box 1</span> teks lanjutan <span class="inline-demo">Box 2</span> <style> .inline-demo { display: inline; background: #e74c3c; padding: 10px 15px; margin: 5px; /* Top/bottom tidak bekerja! */ } </style>

3️⃣ DISPLAY: INLINE-BLOCK

Pengertian Inline-Block

display: inline-block adalah kombinasi TERBAIK dari block dan inline. Elemen mengambil hanya lebar yang dibutuhkan (seperti inline) TAPI bisa diatur width/height (seperti block).

display: inline-block;

Karakteristik Inline-Block:

  • ✅ Hanya mengambil lebar sesuai konten (atau width yang ditentukan)
  • ✅ Bisa berdampingan dengan elemen lain di baris sama
  • ✅ Width & height BISA diatur ✨
  • ✅ Margin & padding bekerja di semua sisi ✨
  • ⚠️ Ada space kecil antar elemen (karena dianggap inline)

Gunakan Kapan?

  • Menu horizontal dengan button
  • Badge atau tag yang ingin ditampilkan berdampingan
  • Card mini yang ingin multiple per baris
  • Tombol-tombol yang ingin sejajar

Kode HTML:

<div class="inline-block-demo">Box 1</div> <div class="inline-block-demo">Box 2</div> <div class="inline-block-demo">Box 3</div> <style> .inline-block-demo { display: inline-block; width: 150px; /* Bisa diatur! */ height: 80px; /* Bisa diatur! */ padding: 20px; /* Bekerja semua sisi */ margin: 10px; /* Bekerja semua sisi */ vertical-align: top; /* Align vertical */ } </style>

4️⃣ DISPLAY: NONE

Pengertian None

display: none menyembunyikan elemen sepenuhnya dari halaman. Elemen tidak ditampilkan DAN tidak mengambil ruang di layout.

display: none;

Karakteristik None:

  • ✅ Elemen tidak terlihat
  • ✅ Elemen tidak mengambil ruang apapun di layout
  • ✅ Cocok untuk show/hide dengan JavaScript
  • ℹ️ Berbeda dengan visibility: hidden (tetap mengambil ruang)

Demo Visualisasi:

display: none vs block

Teks sebelum elemen hidden

🔒 Elemen yang bisa disembunyikan dengan display: none

Teks sesudah elemen hidden - jika elemen hidden, tidak ada gap di sini!

Klik tombol di atas untuk toggle elemen. Perhatikan tidak ada space ketika tersembunyi!

Kode HTML & JavaScript:

<button onclick="toggleHidden()">Toggle Hidden</button> <div id="hiddenElement" class="hidden-demo"> Elemen yang bisa disembunyikan </div> <script> function toggleHidden() { const elem = document.getElementById('hiddenElement'); if (elem.style.display === 'none') { elem.style.display = 'block'; } else { elem.style.display = 'none'; } } </script> <style> .hidden-demo { display: none; /* Tersembunyi awal */ } </style>

Perbedaan: display: none vs visibility: hidden

Property Terlihat? Mengambil Ruang? Gunakan Ketika...
display: none ❌ Tidak ❌ Tidak Ingin elemen hilang total (show/hide)
visibility: hidden ❌ Tidak ✅ Ya (ada space kosong) Ingin elemen tidak terlihat tapi tetap reserved space

5️⃣ DISPLAY: FLEX - Layout 1D

Pengertian Flex (Singkat)

display: flex membuat elemen anak berdampingan secara otomatis (seperti inline-block tapi lebih powerful). Cocok untuk navbar, button groups, atau layout 1 baris.

display: flex; gap: 15px; /* Jarak antar item */

6️⃣ DISPLAY: GRID - Layout 2D

Pengertian Grid (Singkat)

display: grid membuat layout 2 dimensi dengan baris DAN kolom. Cocok untuk dashboard, gallery foto, atau layout kompleks.

display: grid; grid-template-columns: repeat(3, 1fr); /* 3 kolom */ gap: 15px;

📋 Perbandingan Semua Display Values

Display Value Lebar Berdampingan? Width/Height? Gunakan Ketika...
block 100% ❌ Tidak (baris baru) ✅ Ya Container, section, paragraph
inline Sesuai konten ✅ Ya ❌ Tidak Teks biasa, link, span
inline-block Sesuai konten ✅ Ya ✅ Ya Button, badge, menu items
none - - - Sembunyikan elemen total
flex Berbeda ✅ Ya (fleksibel) ✅ Ya Navbar, button groups, 1 baris
grid Berbeda ✅ Ya (terstruktur) ✅ Ya Gallery, dashboard, layout kompleks

💡 Praktik Umum di Website

Struktur Website Typical:

<header> { display: block; } /* Full width */ <nav> { display: flex; } /* Menu items horizontal */ <main> { display: block; } /* Full width */ <article> { display: block; } <sidebar> { display: block; } <footer> { display: block; } /* Full width */ <ul> { display: flex; } /* Footer links horizontal */

Contoh Kombinasi:

/* Navigation Bar */ .navbar { display: flex; gap: 20px; } .nav-item { display: inline-block; padding: 10px; } /* Card Grid */ .card-container { display: grid; grid-template-columns: repeat(3, 1fr); } .card { display: block; } /* Product Listing */ .products { display: flex; flex-wrap: wrap; } .product { display: inline-block; width: 23%; margin: 1%; }
🎯 Golden Rules:
  • Gunakan block untuk container utama
  • Gunakan flex untuk layout 1 baris (nav, buttons)
  • Gunakan grid untuk layout 2D kompleks
  • Gunakan inline-block untuk element kecil berdampingan
  • Jangan lupa gap untuk jarak otomatis di flex/grid
