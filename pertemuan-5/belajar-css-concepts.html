1️⃣ CSS GRID - Layout 2 Dimensi

Pengertian CSS Grid

CSS Grid adalah sistem layout 2 dimensi yang membagi website menjadi kolom dan baris. Berbeda dengan Flexbox yang hanya 1 dimensi, Grid bisa mengatur baris DAN kolom sekaligus.

Kapan Pakai Grid?
  • Layout dashboard dengan banyak komponen
  • Gallery foto dengan baris & kolom
  • Layout website kompleks
  • Kalender atau data table yang responsif

Contoh 1: Grid 2 Kolom

grid-template-columns: 1fr 1fr;

1fr = 1 fraction (bagian). 1fr 1fr = 2 kolom dengan lebar sama besar

Contoh 2: Grid 3 Kolom

grid-template-columns: 1fr 1fr 1fr; /* atau shorthand: */ grid-template-columns: repeat(3, 1fr);

Contoh 3: Grid 4 Kolom dengan repeat()

grid-template-columns: repeat(4, 1fr); /* repeat(berapa kali, ukuran) */

Contoh 4: Grid dengan Template Areas (Advanced)

grid-template-columns: repeat(3, 1fr); grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; .grid-item.header { grid-area: header; } .grid-item.sidebar { grid-area: sidebar; } .grid-item.main { grid-area: main; } .grid-item.footer { grid-area: footer; }

Gap (Jarak antar item)

gap: 15px; /* Jarak 15px antar item */

Membuat jarak otomatis antara semua item dalam grid.

2️⃣ LINEAR GRADIENT - Warna Gradual

Pengertian Linear Gradient

linear-gradient adalah fungsi CSS untuk membuat warna yang berubah secara gradual dari satu warna ke warna lain. Hasilnya lebih menarik daripada warna solid!

Contoh 1: Gradient Sederhana (Kanan)

background: linear-gradient(to right, #667eea, #764ba2);
to right: Biru → Ungu

Contoh 2: Gradient dengan Sudut 45 Derajat

background: linear-gradient(45deg, #f093fb, #f5576c);
45deg: Rosa → Merah

Contoh 3: Gradient dengan 3 Warna

background: linear-gradient(to bottom, #667eea, #764ba2, #f093fb);
to bottom: 3 Warna

Contoh 4: Gradient dengan Posisi Stop

background: linear-gradient( 90deg, #667eea 0%, #667eea 25%, #764ba2 50%, #f093fb 75%, #f5576c 100% );

Bisa mengontrol di mana warna berubah dengan persentase atau px

Banyak Stop Point

Contoh 5: Radial Gradient (dari tengah)

background: radial-gradient(circle, #667eea, #764ba2);
Radial dari Tengah
Syntax Umum:
linear-gradient(arah, warna1, warna2)
  • Arah: to right, to bottom, 45deg, dll
  • Warna: #hexcode, rgb(), atau color names
  • Stop: Opsional - untuk kontrol posisi warna

3️⃣ TRANSFORM - Pergerakan & Perubahan Bentuk

Pengertian Transform

Transform mengubah posisi, ukuran, atau rotasi elemen tanpa mempengaruhi layout website. Sering digabung dengan transition untuk efek smooth.

Contoh 1: translateY (Naik/Turun)

transform: translateY(-10px); /* translateY(positif) = turun, translateY(negatif) = naik */

Hover ke box di bawah untuk melihat efeknya:

Hover ke saya → Naik! 👆

Contoh 2: scale (Membesar/Mengecil)

transform: scale(1.1); /* 1.1 = 110% dari ukuran awal */
Hover → Membesar! 🔍

Contoh 3: rotate (Memutar)

transform: rotate(5deg); /* Positif = searah jarum jam */
Hover → Berputar! 🔄

Contoh 4: Kombinasi Multiple Transform

transform: translateY(-10px) scale(1.05) rotate(2deg);

Bisa menggabung beberapa transform sekaligus!

Hover → Semua Transform! 🎯
Transform Values:
  • translateX(px) - geser kanan/kiri
  • translateY(px) - geser atas/bawah
  • scale(angka) - ubah ukuran (1 = normal)
  • rotate(deg) - putar (positif = searah jarum jam)
  • skew(deg) - miringkan

Penting: Tambahkan Transition!

transition: all 0.3s ease; /* Buat perubahan smooth daripada instant */

4️⃣ CSS SELECTORS - nth-child & nth-of-type

Pengertian Selector

Selector adalah cara memilih elemen HTML tertentu untuk diberikan CSS. :nth-child() memilih elemen berdasarkan posisinya di antara siblings.

Contoh 1: nth-child(odd) - Baris Ganjil

tr:nth-child(odd) { background-color: #f5f5f5; } /* odd = 1, 3, 5, 7 (baris ganjil) */

Lihat tabel di bawah, baris nomor 1, 3, 5 memiliki warna abu-abu:

No Nama Status
1 Baris Ganjil Background Abu-abu
2 Baris Genap Background Putih
3 Baris Ganjil Background Abu-abu
4 Baris Genap Background Putih
5 Baris Ganjil Background Abu-abu

Contoh 2: nth-child(even) - Baris Genap

tr:nth-child(even) { background-color: white; } /* even = 2, 4, 6, 8 (baris genap) */

Contoh 3: nth-child(3n) - Setiap 3 Elemen

tr:nth-child(3n) { border-left: 4px solid blue; } /* 3n = 3, 6, 9, 12 (setiap 3 elemen) */

Lihat tabel di atas, baris 3 memiliki border biru di kiri

Contoh 4: first-child, last-child, nth-child

li:first-child { /* Item pertama */ } li:last-child { /* Item terakhir */ } li:nth-child(2) { /* Item nomor 2 */ } li:nth-of-type(3) { /* Item ketiga dari type yang sama */ }
  • ✅ Item Pertama (first-child) - Hijau
  • 🟨 Item Kedua (nth-child(2)) - Kuning
  • 🟪 Item Ketiga (nth-of-type(3)) - Ungu
  • Item Keempat
  • 🔴 Item Terakhir (last-child) - Merah
Selector Cheat Sheet:
  • :first-child - Item pertama
  • :last-child - Item terakhir
  • :nth-child(n) - Item ke-n
  • :nth-child(odd) - Item ganjil (1,3,5...)
  • :nth-child(even) - Item genap (2,4,6...)
  • :nth-child(3n) - Setiap 3 item (3,6,9...)
  • :nth-child(2n+1) - Item ganjil (formula)
  • :nth-of-type(n) - Item ke-n dari tag yang sama

5️⃣ REAL WORLD - Kombinasi Semua Konsep

Membuat Card Grid (Seperti Landing Page)

.card-grid { display: grid; /* Gunakan Grid */ grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; } .card { background: white; transition: all 0.3s ease; } .card:hover { transform: translateY(-10px); /* TranslateY saat hover */ box-shadow: 0 10px 30px rgba(0, 0, 0, 0.2); } .card-image { height: 200px; background: linear-gradient(135deg, #667eea, #764ba2); }

📋 Summary - Kapan Pakai Apa?

Konsep Gunakan Ketika... Contoh
CSS Grid Buat layout kompleks dengan baris & kolom Dashboard, gallery, portfolio
Linear Gradient Ingin background dengan warna gradual Hero section, card background
TranslateY Ingin elemen bergerak atas/bawah Hover effect pada card, button
nth-child(odd) Alternate warna di tabel atau list Zebra striping di tabel
💡 Pro Tips:
  • Selalu tambah transition untuk transform agar smooth
  • Gunakan gap di grid daripada margin untuk konsistensi
  • Gradient bisa pakai di background image, tidak hanya solid color
  • Kombinasikan multiple selector untuk presisi lebih
  • Test responsive design di browser DevTools (F12)
