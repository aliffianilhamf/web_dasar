1. Pendahuluan

Selamat datang di modul interaktif lengkap mengenai CSS Padding dan Margin. Setiap elemen HTML pada halaman web pada dasarnya adalah sebuah kotak (box). Memahami bagaimana kotak-kotak ini diberi ruang adalah kunci utama dalam web development.

Apa itu CSS Box Model?

Box Model adalah konsep dasar dalam CSS yang membungkus setiap elemen HTML. Terdiri dari margin, border, padding, dan content aktual.

Mengapa Spacing Penting?

  • Meningkatkan keterbacaan konten (readability).
  • Memberikan hierarki visual yang jelas.
  • Mencegah elemen terlihat menumpuk dan berantakan.

Kesalahan Umum Pemula

Pemula sering kali menebak-nebak antara menggunakan padding atau margin, yang berujung pada layout yang hancur saat ukuran layar berubah.

2. Mengenal CSS Box Model

Setiap elemen web terdiri dari 4 layer utama dari dalam ke luar:

  1. Content: Isi elemen (teks, gambar).
  2. Padding: Ruang transparan di dalam elemen, antara content dan border.
  3. Border: Garis batas elemen.
  4. Margin: Ruang transparan di luar elemen, memisahkan elemen ini dari elemen lain.

3. Padding

Definisi Padding

Padding adalah ruang kosong di area dalam elemen, tepatnya di antara konten dan batas (border). Padding digunakan ketika Anda ingin melapangkan isi dari tepi kotaknya, dan padding mengambil warna background yang sama dengan elemen tersebut.

Sintaks Dasar & Per Sisi

.box {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;
}

Shorthand Padding

CSS menyediakan singkatan (shorthand) searah jarum jam (Top, Right, Bottom, Left):

  • padding: 20px; (Semua sisi 20px)
  • padding: 20px 40px; (Atas-Bawah 20px, Kiri-Kanan 40px)
  • padding: 20px 40px 10px; (Atas 20px, Kiri-Kanan 40px, Bawah 10px)
  • padding: 20px 40px 10px 5px; (Atas, Kanan, Bawah, Kiri)

Contoh Implementasi HTML & CSS

Cobalah ketik kode berikut di file terpisah untuk melihat efek padding secara langsung:

<style>
  .card-padding {
    background-color: #bfdbfe;
    padding: 30px;
    border: 2px solid #3b82f6;
  }
</style>

<div class="card-padding">
  Teks di dalam kotak ini didorong sejauh 30px dari border karena padding.
</div>

4. Margin

Definisi Margin

Margin adalah ruang di sebelah luar elemen (di luar border). Margin menolak elemen lain menjauh. Margin selalu transparan dan tidak ikut diwarnai oleh background elemen.

Shorthand Margin

Sama seperti padding, dibaca searah jarum jam:

.elemen {
    margin: 10px 20px 30px 40px; /* Top Right Bottom Left */
}

Contoh Implementasi HTML & CSS

Kode di bawah ini menunjukkan bagaimana margin memisahkan dua elemen HTML agar tidak saling menempel:

<style>
  .box-margin {
    background-color: #fca5a5;
    padding: 15px;
    margin-bottom: 25px;
  }
</style>

<div class="box-margin">Kotak Pertama</div>
<div class="box-margin">Kotak Kedua berjarak 25px dari kotak pertama</div>

Margin Auto & Centering

margin: auto; sering digunakan pada blok elemen (yang memiliki width) untuk memposisikannya di tengah horizontal secara otomatis.

5. Perbedaan Padding vs Margin

Aspek Padding Margin
Lokasi Di dalam Border Di luar Border
Efek Background Mengikuti background elemen Transparan (Background parent)
Fungsi Utama Membesarkan ruang dalam, menjauhkan teks dari pinggir Mendorong elemen lain agar tidak berdempetan
Nilai Negatif Tidak Diizinkan Diizinkan (Negative Margin)

6. Margin Collapse

Margin Collapse adalah kondisi di mana margin vertikal (top dan bottom) dari dua elemen yang bersentuhan bergabung menjadi satu margin yang nilainya sama dengan margin terbesar (bukan dijumlahkan).

<style>
  .box1 { 
    margin-bottom: 50px; 
    background-color: #fde047;
  }
  .box2 { 
    margin-top: 30px; 
    background-color: #bbf7d0;
  }
</style>

<div class="box1">Box Pertama</div>
<div class="box2">Box Kedua</div>

Jarak akhir antara box1 dan box2 adalah 50px, bukan 80px.

16. Latihan Praktik: Membuat Card Profile

Tugas Anda adalah mempraktikkan teori yang sudah dipelajari dengan membuat sebuah Card Profile mandiri. Ketiklah HTML dan CSS berikut dan bereksperimen dengan mengubah nilai padding dan margin-nya.

<!DOCTYPE html>
<html>
<head>
  <style>
    body {
      background-color: #e2e8f0;
      margin: 0;
      padding: 50px; /* Jarak agar card tidak menyentuh layar */
    }
    .card {
      background-color: white;
      width: 300px;
      margin: 0 auto; /* Card di-center secara horizontal */
      padding: 25px;  /* Teks dan tombol tidak menempel ke border */
      border-radius: 12px;
      box-shadow: 0 4px 6px rgba(0,0,0,0.1);
      text-align: center;
    }
    .card h3 {
      margin-top: 0;
      margin-bottom: 10px; /* Jarak dengan teks profesi */
    }
    .card p {
      margin-bottom: 20px; /* Jarak antara teks profesi dan tombol */
      color: #64748b;
    }
    .btn-contact {
      display: inline-block;
      padding: 10px 20px; /* Ruang di dalam tombol */
      background-color: #2563eb;
      color: white;
      text-decoration: none;
      border-radius: 6px;
    }
  </style>
</head>
<body>
  <div class="card">
    <h3>Ahmad Fulan</h3>
    <p>Frontend Developer</p>
    <a href="#" class="btn-contact">Hubungi Saya</a>
  </div>
</body>
</html>

17. Ringkasan Akhir

  • Padding untuk ruang dalam, Margin untuk ruang luar.
  • Gunakan margin: 0 auto; untuk centering div.
  • Hati-hati dengan Margin collapse pada aliran vertikal antar blok.
  • Pada Flexbox dan Grid moderen, properti gap sangat direkomendasikan untuk menggantikan fungsi margin antar elemen.
