1. CSS Grid Layout

CSS Grid Layout adalah sistem tata letak paling kuat dan komprehensif di dalam CSS. Perbedaan utamanya dengan Flexbox terletak pada dimensi pengaturannya.

  • Flexbox: Tata letak satu dimensi (Satu sumbu). Dirancang untuk mengatur baris ATAU kolom. Cocok untuk komponen skala kecil (navigasi, susunan tombol, daftar kartu).
  • CSS Grid: Tata letak dua dimensi. Dirancang untuk mengatur baris DAN kolom secara bersamaan. Sangat tepat digunakan untuk membagi ruang pembentuk struktur keseluruhan halaman web.

2. Konsep Dasar dan Deklarasi

Sama halnya dengan Flexbox, penggunaan Grid ditetapkan pada elemen kontainer (induk) yang berfungsi membungkus item-item (anak) di dalamnya.

.container {
    display: grid;
}

Mendeklarasikan display: grid; belum akan memperlihatkan perubahan tata letak yang signifikan pada anak elemen sebelum Anda mendefinisikan proporsi kolom dan barisnya.

3. Pengaturan Kolom dan Baris

Konfigurasi bentuk Grid dilakukan melalui dua properti utama untuk membelah kontainer menjadi beberapa trek.

  • grid-template-columns: Menentukan jumlah dan lebar kolom pada tata letak grid.
  • grid-template-rows: Menentukan jumlah dan tinggi baris pada tata letak grid.
.grid-container {
    display: grid;
    grid-template-columns: 200px 300px 200px;
    grid-template-rows: 100px 150px;
    gap: 15px; /* Jarak antar kolom dan baris */
}

Berdasarkan kode di atas, terbentuk sebuah kontainer dengan 3 kolom (lebar masing-masing 200px, 300px, 200px) dan 2 baris (tinggi 100px dan 150px).

4. Satuan Fraction (fr) dan Fungsi repeat()

Penggunaan nilai px statis dalam grid sangat tidak disarankan untuk layout responsif. CSS Grid memperkenalkan satuan Fraction (fr).

Satu fr berarti satu bagian proporsional dari sisa ruang yang tersedia di kontainer grid.

Contoh Penggunaan fr:

.grid-container {
    /* Kolom pertama mengambil 1 bagian, kolom kedua 2 bagian */
    grid-template-columns: 1fr 2fr;
}

Fungsi repeat()

Untuk jumlah kolom yang banyak dengan ukuran yang seragam, sangat tidak efisien menulisnya satu per satu. Penggunaan repeat() dapat mempersingkat kode.

.grid-container {
    /* Menggantikan penulisan: 1fr 1fr 1fr 1fr */
    grid-template-columns: repeat(4, 1fr);
}

5. Membentangkan dan Memposisikan Item

Properti grid bukan hanya tentang kontainer. Anda bisa memanipulasi rentang ukuran elemen anak agar membentang melewati beberapa kolom atau baris menggunakan properti grid-column dan grid-row.

Garis pembatas grid (Grid Lines) dihitung dari awal angka 1. Sebagai contoh, jika Anda memiliki 3 ruang kolom, maka Anda memiliki 4 Garis (Line 1 di paling kiri, Line 4 di paling kanan).

.header {
    /* Memulai item dari garis kolom 1 hingga garis kolom 4 (membentang penuh 3 kolom) */
    grid-column: 1 / 4; 
}

.sidebar {
    /* Membentangkan elemen ke bawah mencakup 2 ruang baris berturut-turut */
    grid-row: span 2; 
}

7. Latihan Praktik Layout Halaman

Ini adalah contoh terapan standar struktur kerangka web. Menggunakan Grid, pembagian ruang untuk Header, Sidebar, Konten Utama, dan Footer menjadi sangat teratur.

<style>
  .layout-dasar {
    display: grid;
    grid-template-columns: 250px 1fr; /* Kiri tetap 250px, Kanan mengisi sisa */
    grid-template-rows: auto 1fr auto; /* Atas dan bawah sesuaikan isi, Tengah menyesuaikan sisa layar */
    gap: 10px;
    min-height: 400px;
  }
  
  .layout-dasar > div {
    padding: 20px;
    background-color: #f1f5f9;
    border: 1px solid #94a3b8;
  }

  .kepala {
    grid-column: 1 / 3;  /* Membentangkan header ke semua area kolom */
    background-color: #cbd5e1;
    text-align: center;
  }
  
  .kaki {
    grid-column: 1 / 3;  /* Membentangkan footer ke semua area kolom */
    text-align: center;
  }
</style>

<div class="layout-dasar">
  <div class="kepala">Header (Logo + Menu)</div>
  <div class="samping">Sidebar (Sisi menu)</div>
  <div class="tengah">Konten Utama Web (Artikel)</div>
  <div class="kaki">Footer Web</div>
</div>

Praktekkan kode tersebut di halaman terpisah dan saksikan efektivitas sistem tata letak grid dua dimensi.

9. Ringkasan Akhir

  • CSS Grid merupakan solusi tata letak paling mumpuni untuk kerangka halaman (dua dimensi: baris sekaligus kolom).
  • Pengaturan ruang horizontal maupun vertikal dituntaskan pada kontainer induk melalui properti grid-template-columns dan grid-template-rows.
  • Unit pecahan atau proporsional (fr) adalah standar terukur untuk sistem responsive modern menggantikan penggunaan satuan piksel kaku.
  • Penggunaan grid-column memberikan kontrol pelebaran per-cell pada elemen spesifik di dalam struktur Grid.
