1. Flexbox (Flexible Box)

Flexbox adalah modul tata letak CSS yang dirancang khusus untuk mengatur penempatan (layout) pada antarmuka web, khususnya menata deretan elemen dalam satu dimensi (satu baris mendatar OR satu kolom menurun).

Hadirnya Flexbox menggantikan metode lama yang rumit (seperti *float* dan perhitungan rumit *margin*), menjadikannya standar wajid industri web modern.

Kunci memahami Flexbox terletak pada pemahaman hubungan antara bagian Induk (Flex Container) dan bagian Anak (Flex Items) di dalamnya.

2. 🗺️ Konsep Sumbu Utama dan Melintang

Di Flexbox, elemen-elemen mengalir pada suatu Sumbu (Axis). Mengetahui arah sumbu ini wajib, karena arah ini akan mendikte cara properti Flexbox bekerja.

  • Main Axis (Sumbu Utama): Arah tempat bernaungnya *Flex Items* disejajarkan. Secara default, arahnya dari KIRI ke KANAN (horizontal).
  • Cross Axis (Sumbu Melintang): Arah yang tegak lurus memotong *Main Axis*. Secara default, arahnya dari ATAS ke BAWAH (vertikal).

3. 🪄 Perintah Ajaib: `display: flex;`

Semua kesaktian Flexbox dimulai dengan menyuntikkan properti display: flex; ke elemen pembungkusnya (parent). Begitu diaktifkan, semua elemen keturunan langsungnya secara otomatis berubah sejajar dan rapi.

<style>
  .container {
    display: flex;
    background-color: #e2e8f0;
    padding: 10px;
  }
  .item {
    background-color: #3b82f6;
    color: white;
    padding: 20px;
    margin: 5px;
  }
</style>

<div class="container">
  <div class="item">Anak 1</div>
  <div class="item">Anak 2</div>
  <div class="item">Anak 3</div>
</div>

Hanya dengan menyematkan display: flex;, seketika `div` yang tadinya menumpuk dari atas ke bawah (ingat sifat *block*), akan berubah berjajar lurus secara horizontal.

4. Merapikan Barisan: Justify & Align

Dua properti ini yang membuat Flexbox begitu dicintai untuk menengahkan atau mengatur jarak elemen.

justify-content

Mengatur kelurusan item di sepanjang Sumbu Utama (Main Axis) / defaultnya *Horizontal*.

  • flex-start: Rata kiri (default)
  • flex-end: Rata kanan
  • center: Tepat di tengah-tengah
  • space-between: Jarak menyebar merata, menempel ke ujung kiri-kanan.
  • space-around / space-evenly: Jarak menyebar secara adil di sekeliling item.

align-items

Mengatur kelurusan item memotong Sumbu Melintang (Cross Axis) / defaultnya *Vertikal*.

  • stretch: Dipanjangkan memenuhi tinggi kontainer (default).
  • flex-start: Menempel di atas.
  • flex-end: Menempel di dasar kontainer.
  • center: Berada persis di tengah secara vertikal.

5. Memutar Arah & Memberi Jarak

flex-direction

Properti ini mendikte arah Sumbu Utama. Berpengaruh krusial karena ia memutar logic `justify-content`.

Opsi: row (default Kiri-Kanan), row-reverse (Kanan-Kiri), column (Atas-Bawah tegak), column-reverse (Bawah-Atas).

flex-wrap

Fleksibilitas sering kali membuat teks sesak jika dipaksa 1 baris. Dengan properti ini, anak-anak flex diizinkan tumpah ke baris baru seperti teks bacaan (wrap).

Opsi: nowrap (default), wrap.

gap

Berhenti menggunakan margin kanan/kiri yang repot, gunakan gap pada container! Nilainya berlaku murni sebagai jarak kosong ruang antara elemen baik horizontal maupun vertikal.

.flex-parent {
    display: flex;
    gap: 20px; /* Jarak pasti 20px di antara semua item, ajaib! */
}

6. Properti untuk Elemen Anak (Item)

Meski sebagian besar diatur di kontainer Induk, ada satu fungsi kuat yang dimiliki flex item, yaitu flex-grow / flex.

Properti flex: 1; pada sebuah anak (atau porsi `flex: 2;` untuk porsi dobel) memerintahkannya untuk menelan, mengambil, meregangkan diri ke seluruh sisa ruang yang menganggur dari baris tersebut.

8. Latihan Praktik: Navbar Sederhana Flexbox

Salah satu penggunaan flexbox paling populer adalah membuat bagian Header Web (Navbar). Kita ingin Logo di ujung kiri, dan kumpulan menu ada di ujung sebelah kanan, sejajar teratur secara vertikal. Flexbox `space-between` menyelamatkan kita!

<style>
  header {
    display: flex;
    justify-content: space-between; /* Lempar menu ke sisi berseberangan */
    align-items: center;            /* Rata tengah secara gaya vertikal */
    background-color: #1e293b;
    padding: 15px 30px;
    color: white;
  }
  
  .logo {
    font-size: 24px;
    font-weight: bold;
  }
  
  .nav-links {
    display: flex;                  /* Anaknya kontainer pun bisa menjadi Flex Parent lagi! */
    gap: 20px;                      /* Jarak mulus tanpa repot margin */
    list-style: none;
  }

  .nav-links a {
    color: #cbd5e1;
    text-decoration: none;
  }
</style>

<header>
  <div class="logo">WebKu</div>
  <ul class="nav-links">
    <li><a href="#">Home</a></li>
    <li><a href="#">Layanan</a></li>
    <li><a href="#">Kontak</a></li>
  </ul>
</header>

10. Ringkasan Akhir

  • display: flex; dikenakan pada kontainer induk tempat bersemayamnya kotak-kotak anak.
  • justify-content mengatur sejajar anak-anak pada Main Axis. (Geser-geser horizontal).
  • align-items mengendalikan penyejajaran pada Cross Axis. (Cebol-jangkung vertikal).
  • Gunakan gap untuk spasi cepat sempurna antar flex items, dan tinggalkan metode repot berbalut `margin`!
  • Ingatlah, menguasai kombinasi justify-content: center; align-items: center; berarti tidak ada lagi derita menyusun elemen ke tengah halaman (*Absolute Centering* terselesaikan)!
